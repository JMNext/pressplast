# МАТРИКС-ПЛАСТ — Литье пластмасс под давлением в СПБ

Данный репозиторий содержит демонстрационную версию (лендинг-пейдж) официального сайта компании **«МАТРИКС-ПЛАСТ»** (г. Санкт-Петербург), специализирующейся на услугах промышленного литья изделий из пластмасс под давлением на термопластавтоматах (ТПА).

---

## 🎨 Концепция дизайна
Сайт выполнен в выразительном стиле **цифрового нео-брутализма (Neo-Brutalism)** с элементами немецкой школы **Баухаус**:
* **Сетка и границы**: Использование жестких Grid-сеток с контрастными черными разделителями (`2px solid #000`).
* **Палитра**: Минималистичная индустриальная палитра с текстурным фоном серого бетона (`#dfdfdf`), глубоким черным, чистым белым и ярким сигнальным оранжевым акцентом (`#ff5500`) для элементов призыва к действию (CTA).
* **Типографика**: Использование плотного гротеска **Inter** для крупных массивных заголовков и моноширинного шрифта **Roboto Mono** для вывода технических характеристик, кодов сырья и результатов расчетов.
* **Интерактивные микро-эффекты**: Резкие тактильные смещения кнопок и карточек при наведении (`translate(-4px, -4px)`) с жесткой черной тенью вместо мягких градиентных теней и размытий.

---

## 🛠️ Технологический стек
Проект реализован без использования сторонних фреймворков и тяжелых библиотек для достижения максимальной производительности, чистоты кода и мгновенной загрузки:
* **HTML5**: Семантическая разметка с валидной иерархией тегов.
* **CSS3 (Vanilla CSS)**: Модульная сетка CSS Grid, Flexbox, переменные (CSS Custom Properties) для дизайн-системы, CSS-фильтры для монохроматизации изображений на лету.
* **JavaScript (ES6 Vanilla JS)**: Интерактивная логика калькулятора, табы материалов, обработка формы отправки и модальные окна.
* **Внешние API**: Яндекс.Карты для схемы проезда (с примененным к iframe CSS-фильтром `grayscale` для поддержки бруталистической эстетики).

## 📂 Структура и состав лендинга
Сайт представляет собой классический одностраничный лендинг с логическим разбиением на технические зоны:
1. **Шапка сайта (Header)**: Название компании, сквозная навигация по разделам, контактный телефон и кнопка быстрого расчета.
2. **Главный экран (Hero)**: Главное УТП, ключевые метаданные (локация, тиражи, точность) и высококачественное монохромное изображение термопластавтомата ТПА с наложенной координатной сеткой.
3. **Преимущества (Advantages)**: Блок с карточками технических возможностей завода (тиражи от 10 штук, полный цикл, контроль качества ОТК).
4. **Сырье и материалы (Materials)**: Интерактивный каталог полимеров (PP, ABS, PA, PC, PE) с переключением технических паспортов характеристик (плотность, рабочая температура, усадка, сферы применения).
5. **Этапы работы (Process)**: Ступенчатая схема движения заказа от анализа чертежа до серийной отгрузки готовой партии.
6. **Калькулятор стоимости и CTA-форма**:
   - Динамический расчет ориентировочной стоимости одной отливки и тиража в зависимости от объема партии (слайдер от 10 до 50 000 шт.), типа полимера и наличия пресс-формы.
   - Перфокарта-заявка для отправки технического задания с возможностью прикрепить файл чертежа и получить подтверждающее модальное окно с ID запроса (`MP-XXXXXX`).
7. **Контакты и Карта (Contacts & Map)**: Юридические реквизиты, график работы, приглашение на производство и интерактивная карта.
8. **Подвал (Footer)**: Правовая информация, ИНН/ОГРН и бруталистический штрих-код.

## 🚀 Запуск проекта
Проект является статическим и не требует сборки.
Для локального запуска достаточно открыть файл `index.html` в любом браузере или запустить простой сервер в корневой папке проекта:

```bash
# Использование Python 3 для локального веб-сервера
python3 -m http.server 8000
```
После чего открыть в браузере: `http://localhost:8000/`.
